🌤️ SkyCast

SkyCast is a simple weather app built with React that shows real-time weather data for any city using the OpenWeatherMap API.

🚀 Features
Search weather by city
Real-time temperature updates
Weather conditions with icons
Responsive UI

🛠️ Tech Stack
React (Vite)
Axios
OpenWeatherMap API
CSS

▶️ Run Locally
git clone https://github.com/calebtolorunleke/skycast.git
cd skycast
npm install
npm run dev

🔑 API Setup

Create a .env file:

VITE_WEATHER_API_KEY=your_api_key_here
